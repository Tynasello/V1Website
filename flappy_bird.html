Flappy Bird (Recreation)

The code was written in python 3.6 with the use of a conda environment and Pygame 1.9.6. Pygame is a set of modules created for the purpose of making video games and multimedia programs in python. The tutorials on realpython.com and pythonprogramming.net assisted me in using pygame effectively. The video shows the running of my main.py file and me playing the game. Pressing the space bar causes the bird to jump. The goal is to get through as many pipes as possible without making contact. The game ends when the bird hits a pipe.
